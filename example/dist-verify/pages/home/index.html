Lattice Demo Site

This example is intentionally compact, but it covers the full feature breadth needed for a hackathon demo.

Start with typed-content-tour for YAML frontmatter, syntax highlighting, table of contents, footnotes, tags, and JSON-LD metadata. Then read template-composition for the custom layout and partial setup, or jump to about for the second collection.

The posts collection is paginated with two published entries per page, and future-work is marked as a draft so it only appears when you build with --drafts.

Built to demonstrate structural frontmatter, content collections, and typed site output.
